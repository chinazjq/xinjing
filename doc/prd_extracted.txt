1. 产品概述


1.1 产品定位

「心镜」是一款集合九大主流人格/性格测试体系的微信小程序。产品核心逻辑是：用户在沉浸式答题后，通过观看激励视频广告解锁完整人格报告，同时精美的结果页驱动用户主动分享到朋友圈，形成社交裂变闭环。产品通过微信流量主（IAA模式）实现广告变现。




1.2 核心价值主张


权威性：九大测试均基于学术/心理学理论，题目经过信效度验证，结果解读严谨且有深度


多样性：从严肃的MBTI、九型人格到趣味的SBTI、恋爱脑测试，覆盖不同使用场景与情绪需求


美观性：结果页视觉设计达到「值得截图发朋友圈」的标准，是裂变的核心驱动力


轻量性：用完即走，无需注册，打开即测，符合微信小程序使用习惯

4. 核心功能需求


4.1 首页与测试选择


4.1.1 测试卡片展示

首页以卡片宫格形式展示全部9个测试模块，每张卡片包含以下信息：


测试名称（中英文双语）


测试类型标签（如：性格分析 / 职业向 / 趣味向 / 情感向）

热度标识（基于历史测试人次实时更新）


测试类型图标（每个测试拥有独立视觉符号）




4.1.2 筛选与排序


分类筛选 Tag： 全部 / 性格类 / 职业类 / 情感类 / 趣味类


排序方式： 默认（推荐）/ 最热 / 最新 / 题量由少到多


搜索入口： 顶部搜索框，支持测试名称和关键词检索




4.1.3 今日推荐 Banner

首页顶部轮播 Banner，运营可配置，用于重点推广当前热门测试（如 SBTI 爆火期间置顶推广）。

4.2.1 测试介绍页


测试背景与理论来源（权威引用，增强信任感）


测试题量、预计用时、结果维度说明


注意事项：建议选第一直觉，无对错之分

4.3 激励广告流程（核心变现链路）

⚡ 核心设计原则

用户必须在完成最后一道题、点击「查看结果」后才触发广告，

此时沉没成本最大，用户流失率最低。绝对不在答题中途插入广告。




4.3.1 广告弹出逻辑


用户答完所有题目，点击「查看我的结果」按钮


页面切换：背景展示结果页模糊化预览（高斯模糊 + 蒙层），前景展示广告卡片


广告卡片展示：&quot;解锁完整报告 · 观看X秒广告 · 完全免费&quot;


调用微信流量主激励视频广告组件，广告播放中不可跳过


广告播放完毕，倒计时结束，页面平滑过渡到完整结果页




4.3.2 广告中间页视觉规范


背景层： 结果页截图 + 高斯模糊（blur: 12px）+ rgba(0,0,0,0.5)蒙层，制造&quot;结果就在眼前&quot;的好奇感


广告卡片： 居中白色圆角卡片，内含广告位 + 进度条 + 倒计时数字


文案： &quot;再等Xs，解锁你的专属人格报告&quot; —— 倒计时实时刷新


强制观看： 广告播放期间关闭按钮不可点击，倒计时归零后自动跳转


失败处理： 广告加载失败时，提示&quot;网络异常，是否重试&quot;，不强制用户付费

4.4 结果页


4.4.1 结果页必含模块


人格类型标题卡： 类型名称、标志性图标/动物/颜色，全屏视觉冲击


核心描述段落： 3–4句话精准刻画该类型，语气既权威又共情，读后要有&quot;这说的就是我&quot;之感


维度雷达图/条形图： 各维度得分可视化（由前端渲染，非图片），支持滑动查看详情


优势与盲区： 该类型在职场、情感、人际中的典型优势与易踩的坑


名人同类型参考： 列举2–3位该类型名人（增强趣味性与认同感）


相配/相冲类型： 该类型与其他类型的相性分析


再测入口： &quot;测测其他方向&quot; 入口，引导用户探索更多测试




4.4.2 结果页 Banner 广告

结果页底部固定展示 Banner 广告（高度48px），用户浏览时持续曝光，不影响内容阅读体验。




4.4.3 分享结果按钮

结果页底部固定「分享给朋友」按钮（绿色主色，高度56px，圆角）。点击后进入分享海报生成页，支持：


分享到微信好友 / 群聊


保存图片到相册（用于发朋友圈）


复制小程序链接




4.5 历史记录与个人中心


测试历史列表： 按时间倒序展示已完成的测试，显示测试名称、完成时间、结果类型


回看结果： 点击历史记录可直接查看完整结果页（不需再次看广告）


数据本地存储： 优先使用微信云开发存储，关联微信OpenID，换设备可恢复


清除记录： 支持单条或全部清除

海报尺寸：750×1334px（标准微信分享图尺寸）

5. 九大测试模块详细规格


5.1 MBTI 十六型人格

属性

说明

理论来源

Myers-Briggs Type Indicator，基于荣格心理类型理论

题目数量

60题（精简权威版，保留核心区分度题目）

预计时长

8–12 分钟

维度体系

E/I（外向/内向）· S/N（实感/直觉）· T/F（思考/情感）· J/P（判断/感知）

结果数量

16种人格类型（INTJ/ENFP/ISFJ等）

测评难度

中等（需认真思考）




5.1.1 题目设计规范


题型：二选一场景式选项（A vs B），不使用Likert量表，降低认知负担


题目来源： 基于官方MBTI题库精选，结合中国文化场景二次优化（如将西方职场场景替换为中国日常场景）


每维度题数： 每个维度15道题，共4维度×15题=60题


选项文案原则： 两个选项均为合理的正面表述，避免&quot;对与错&quot;暗示，保持中立




5.1.2 评分算法


每题选A记+1（对应该维度正向），选B记-1（对应该维度负向）


每维度15题，得分范围-15到+15，&gt;0则判定为正向类型（如E&gt;0则为E型）


边界值处理：某维度得分在-3到+3区间，结果页显示&quot;你在该维度上较为均衡&quot;


最终输出：4字母组合类型 + 各维度百分比（如 E: 68% | I: 32%）




5.1.3 结果页特殊设计


每种类型拥有专属「动物形象」图标（如INTJ-猫头鹰，ENFP-海豚），增强记忆点


提供与各类型的相性分析（灵魂伴侣类型 + 最大挑战类型）


职场建议：该类型适合的岗位类型（3–5条）


恋爱建议：该类型在情感中的典型行为模式

5.4 依恋风格测试

属性

说明

40题

5.5 霍兰德职业兴趣测试

属性

说明

理论来源

John Holland职业兴趣理论，RIASEC六边形模型（1959年，已广泛应用于职业规划）

题目数量

66题（每种类型11题）

预计时长

8–12 分钟

六种类型

R现实型 / I研究型 / A艺术型 / S社会型 / E企业型 / C传统型

结果格式

输出前三高分类型（如 SAI型）+ 六边形雷达图 + 推荐职业清单




5.5.1 职业推荐数据库

针对每种三字母组合（如SAI、RIC等），预置至少10个推荐职业方向，并标注：


职业名称 + 简要说明


学历门槛参考 / 行业发展前景


高考志愿参考专业（针对学生用户）




5.6 大五人格（OCEAN）

属性

说明

理论来源

Big Five Personality Model，Costa &amp; McCrae (1992)，当代心理学最广泛认可的人格框架

题目数量

50题（每维度10题）

预计时长

6–10 分钟

五大维度

O开放性 / C尽责性 / E外倾性 / A宜人性 / N神经质

结果格式

五维度雷达图 + 百分位解读（如&quot;你的开放性高于85%的测试者&quot;）




5.6.1 量表设计要求


采用5级Likert量表（完全不同意/不同意/中立/同意/完全同意）


包含反向计分题（防止用户无脑选同一选项刷量）


结果使用百分位解读，增强专业感和可信度

10. 非功能性需求


10.1 可用性


服务可用性：≥ 99.5%（月度统计）

云函数鉴权：所有云函数调用校验微信登录态，防止未授权访问




10.3 兼容性


微信基础库版本：≥ 2.10.0（覆盖99%以上微信用户）


机型适配：iPhone SE 至 iPhone 15 Pro Max / Android 屏幕比例 16:9 至 21:9 全部适配


深色模式：完整适配系统深色模式，所有页面在深色模式下视觉正常
