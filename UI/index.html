01 — 设计准则 / PRINCIPLES

拒绝 AI 感Not another gradient.

主张 · TENET 01
克制

以米纸作底、墨色为骨。拒绝 Inter、Roboto、紫渐变等过度通用的 AI 审美；以衬线汉字和编号索引重建产品的文学感与仪式感。

主张 · TENET 02
Editorial rhythm.

借用印刷杂志的版式语言——偏移编号、分区索引、细线分隔、单色大字——让每一屏都像一页可收藏的刊物内页。

主张 · TENET 03
镜

产品名取自"心镜"，故以圆为核心图形语言：测试图标、结果光晕、分享海报上的月轮——圆是镜面，是反射，也是一次次照见。

05 — 小程序兼容 / WECHAT NOTES

审美优先，工程可抵达beauty, but shippable.

尺寸 · VIEWPORT

基于 iPhone 13 / 14 基础宽度（375dp，微信小程序 rpx = 2×px），已适配 iPhone SE ~ Pro Max、Android 16:9 至 21:9 全屏段。所有屏幕高度采用 100vh + safe-area-inset-bottom。

字体 · FONT STRATEGY

正式落地时将 display 层降级为系统衬线字体（iOS 苹方 / Android 思源黑体 + 宋体衬线），关键标题以 PNG/SVG 落字替代，保证视觉不劣化；小程序包体积不受影响。

组件 · COMPONENTS

所有广告位使用微信官方 <ad type="video"> / <ad type="banner"> 组件；分享海报用 Canvas 2D 绘制；雷达图使用 ECharts for 小程序版。

深色 · DARK MODE

主题内置 ink / paper 双套变量，监听 wx.onThemeChange，深色模式下墨与纸互换，情绪色饱和度自动 -8%。

性能 · PERFORMANCE

首屏 < 2s · 题目翻页 < 100ms · 海报生成 < 3s。grain 纹理用 base64 内嵌 SVG（< 1KB），避免图片请求。
